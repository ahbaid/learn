# Big O - Algorithm Complexity

### Complexities (worst to best):
#### Exponentials:
* O(n!) - factorial
* O(2^n) - exponential
* O(n^2) - squared
#### Linear:
* O(n log n) - n log n 
* O(n) - n 
* O(log n) - log n 
* O(1) - 1
